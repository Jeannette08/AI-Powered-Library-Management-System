# 📚 LibraAI: AI-Powered Library Management System

An intelligent solution designed to modernize library operations through AI-driven search and automated workflows.

## 🚀 Key Features
* **AI-Powered Search:** Intelligent book discovery and categorization.
* **QR-Based Transactions:** Seamless borrowing and return process using generated QR codes.
* **Penalty & Reservation Management:** Automated tracking of overdue books and reservation queues.
* **Admin Dashboard:** Comprehensive reporting tools for inventory and student activity.

## 🛠️ Tech Stack
* **Language:** Python
* **Database:** SQL (Structured schema for Users, Books, and Transactions)
* **Tools:** QR Code Generation, requirements.txt for dependency management.


## ⚙️ Installation
1. Clone the repo: `git clone https://github.com/Jeannette08/AI-Powered-Library-Management-System.git`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the system: `python libraai_system.py`
